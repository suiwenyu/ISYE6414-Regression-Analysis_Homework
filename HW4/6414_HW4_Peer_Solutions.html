Background

The owner of a company would like to be able to predict whether employees will stay with the company or leave.

Data Description

The data contains information about various characteristics of employees. Please note that the dataset has been updated to account for repetitions, which is needed for Goodness of Fit Assessment. See below for the description of these characteristics. The data consists of the following variables:

  1. Age.Group: 1-9 (1 corresponds to teen, 2 corresponds to twenties, etc.)
  2. Gender: 1 if male, 0 if female
  3. Tenure: Number of years with the company
  4. Num.Of.Products: Number of products owned
  5. Is.Active.Member: 1 if active member, 0 if inactive member
  6. Staying: Fraction of employees that stayed with the company for a given set of predicting variables

  1. 3 pts - Write down the equation for the odds of staying.

The probability of success (staying) given the predicting variable \(X_{Num.Of.Products2}\) can be defined as:

\[p_{staying} = P(Y=1|X_{Num.Of.Products2})\].

The equation for the odds of staying can be defined as:

\[\frac{p_{staying}}{1-p_{staying}} = e^{\beta_{0} + \beta_{1} * X_{Num.Of.Products2}}\]

Substituting the model parameters with the estimates, we get:

\[\frac{\hat{p}_{staying}}{1-\hat{p}_{staying}} = e^{0.37886 - 1.7668 * X_{Num.Of.Products2}}\]

We will use a chi-square test to compare the fitted model to the null model.

# Calculate overall significance

         (model1$df.null - model1$df.resid))
## [1] 0

The p-value is very close to zero, so we can reject the null hypothesis that the slope coefficient is equal to zero and conclude that the model is significant overall.

  1. 3 pts - Which regression coefficients are significantly nonzero at the 0.01 significance level? Which are significantly negative? Why?

We look at the estimates and p-values given in the model summary. The p-values are associated to the two-sided z-test with the hypotheses: \(H_0:\beta_i=0\) and \(H_a:\beta_i\neq0\)

We can see that the estimated coefficients associated to the intercept and Num.Of.Products2 are significantly non-zero with p-values very close to zero. Hence, both regression coefficients are significantly nonzero at the 0.01 significance level.

  • Option 1:

To test for significantly negative coefficients (\(H_0: \beta_i \geq 0\) and \(H_a: \beta_i < 0\)), we can look at the estimates and p-values given in the model summary.

The estimate for Num.Of.Products2 is negative, and the corresponding p-value is very close to zero. Half of the p-value of Num.Of.Products2 is still less than 0.01, indicating that the coefficient for Num.Of.Products is significantly negative at this significance level.
